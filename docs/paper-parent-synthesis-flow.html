Parent Output: Thesis

RLM-FORGE's supported claim is that Recursive Language Model behavior can be lifted into an existing agent runtime as an enforceable execution contract, preserving bounded sub-calls, trace state, replayable child evidence, provider-swappable execution, and evidence-gated parent synthesis, without claiming a new model architecture or live downstream model-quality improvement.

Supported Contribution Claims

Runtime-Lifted RLM

RLM is presented as a runtime primitive existing agent infrastructure can execute, inspect, and enforce.

Hermes as Inner Execution

Hermes RPC hosts recursive sub-calls and keeps the inner provider swappable.

Ouroboros Control Plane

The scaffold owns bounded context, recursive sub-call graphs, child evidence, and parent synthesis control.

TraceGuard Evidence Gate

Parent claims need accepted child evidence handles before becoming accepted state.

Portability Evidence

The same RLM-FORGE+TraceGuard contract passes a 24-cell matrix across Hermes+GLM, Claude Code, and Codex.

Bounded Benchmark Claim

The live truncation fixture is score parity after claim-aware rescoring, not a recursive quality win.

Memory as Policy

Memory is admitted as schema/retry/routing prior, not as accepted evidence.

Contribution Benchmarks

Deterministic tests support contamination rejection, layered role separation, and adaptive repair efficiency.
